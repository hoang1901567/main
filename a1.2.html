Các thẻ trong HTML

HTML Basic

Cấu trúc cơ bản của một tài liệu HTML bao gồm các thẻ <!DOCTYPE>, <html>, <head>, và thẻ <body> 

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

HTML Elements

Các phần tử HTML là các khối xây dựng nên các trang HTML, được xác định bởi thẻ bắt đầu, nội dung và thẻ kết thúc.

<p>This is an HTML element.</p>

HTML Attributes

Thuộc tính cung cấp thông tin bổ sung về các phần tử HTML và luôn được chỉ định trong thẻ bắt đầu.

<a href="https://www.example.com">Visit Example</a>

HTML Headings

Tiêu đề HTML được định nghĩa bằng thẻ <h1> và <h6> , trong đó thẻ <h1> quan trọng nhất.

<h1>Main Heading</h1>
<h2>Subheading</h2>

HTML Paragraphs

Các đoạn văn được định nghĩa bằng thẻ <p> 

<p>This is a paragraph.</p>

HTML Styles

Kiểu HTML được sử dụng để thêm kiểu cho các thành phần, chẳng hạn như màu sắc, phông chữ và khoảng cách.

<p style="color:blue;">This is a styled paragraph.</p>

HTML Formatting

Các thành phần định dạng bao gồm <b>, <i>, <u>, và các thành phần khác để định dạng văn bản.

<b>Bold Text</b>, <i>Italic Text</i>

HTML Quotations

Trích dẫn trong HTML có thể được sử dụng <blockquote> cho trích dẫn khối và <q>trích dẫn nội tuyến.

<blockquote>This is a block quote.</blockquote>

HTML Comments

Bình luận được thêm vào <!-- --> và không được hiển thị trên trình duyệt.

<!-- This is a comment -->

HTML Colors

Màu HTML có thể được áp dụng bằng cách sử dụng tên, mã hex, RGB hoặc các định dạng màu khác.

<p style="color:#ff6347;">This is a colored paragraph.</p>

HTML Links

Liên kết được tạo bằng thẻ <a>, xác định đích đến bằng href thuộc tính 

<a href="https://www.example.com">This is a link</a>

HTML Images

Hình ảnh được những bằng thẻ <img> , với src thuộc tính chỉ rõ nguồn hình ảnh. 

<img src="image.jpg" alt="Description">

HTML Tables

Bảng được định nghĩa bằng thẻ <table> , sử dụng <tr>, <th>, và <td> cho các hàng và ô.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

HTML Lists

Danh sách có thể được sắp xếp(<ol>) hoặc không có thứ tự (<ul>), với các mục danh sách được xác định bằng cách sử dụng <li>.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
